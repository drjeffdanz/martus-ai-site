# martus.ai

**The Enterprise Intelligence Platform — Witness Every Decision. Across Every System.**

The marketing site for martus.ai: the cross-system witness for the hyperautomated enterprise. Reconstruct causal chains across systems, catch silent AI failures, and ship audit-ready provenance.

## Stack

- Single static `index.html` — no build step, no dependencies.
- Fonts via Google Fonts CDN.

## Local preview

Open `index.html` in any browser, or serve the folder:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Deploy (Vercel)

This is a static site. In Vercel:

- **Framework Preset:** Other
- **Build Command:** *(leave empty)*
- **Output Directory:** `.` (root)

Then add the custom domain `martus.ai` in Project → Settings → Domains.
